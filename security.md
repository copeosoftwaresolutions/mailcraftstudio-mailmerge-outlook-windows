# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in MailCraft Studio, please **do not** open a public GitHub issue.

Contact the developer directly via the [Microsoft Store listing](https://apps.microsoft.com/detail/9N81V7ZQ9ZS8).

We take security reports seriously and will respond as quickly as possible.

## Scope

MailCraft Studio is a local Windows desktop application. It does not transmit data to external servers. Security concerns most relevant to this app include:

- Local file handling vulnerabilities
- CSV import handling
- Email sending behaviour via Outlook/Hotmail/Live

Thank you for helping keep MailCraft Studio safe.

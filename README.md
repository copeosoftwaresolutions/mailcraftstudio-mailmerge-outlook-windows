## What is MailCraft Studio?

**MailCraft Studio** is a powerful native Windows desktop application for building professional email templates and running mail merges with Outlook and Hotmail. Designed for businesses, marketers, and freelancers who want to send personalized email — without paying a monthly subscription.

Unlike web-based email marketing tools like Mailchimp or Constant Contact, MailCraft Studio is a **standalone desktop app**:

- 🆓 **Free to download and use** — core features available at no cost
- 💳 **Upgrade once** — unlock premium features with a one-time in-app purchase, no subscription
- 🔒 **No cloud uploads** — your contacts and templates stay on your machine
- 📶 **Works offline** — no internet connection required to build templates

## FAQ

**Is it really free?**
Yes. MailCraft Studio is free to download with core features available at no cost. A one-time in-app purchase unlocks the full premium feature set — no subscription ever.

**Does it work with Outlook?**
Yes. Built specifically for Outlook, Hotmail, and Microsoft Live accounts.

**Does it require an internet connection?**
No. The app works entirely offline. Your templates and contacts never leave your machine.

**What Windows versions are supported?**
Windows 10 and Windows 11.

**Can I import my contact list?**
Yes. Import contacts via CSV file with automatic field mapping.

**How is it different from Word's mail merge?**
MailCraft Studio is purpose-built for email with a drag-and-drop template builder, mobile preview, and a clean modern interface.

**How is it different from Mailchimp or Constant Contact?**
No subscription, no cloud, no contact limits. Your data stays on your PC.

**What's included in the free version?**
The free version includes the template builder, basic mail merge, Outlook support, CSV import, and mobile preview. Premium unlocks unlimited bulk sending, attachments, CC/BCC, and the full template library.
